# Licensing

IMPACT uses a split license structure to distinguish software from dataset assets and documentation.

Software:
- [LICENSE](/hkfs/work/workspace/scratch/jv8319-HAS/IMPACT_open_source/LICENSE) applies to repository-authored code, scripts, and configuration files, unless otherwise noted

Dataset assets and documentation:
- [LICENSE-DATA](/hkfs/work/workspace/scratch/jv8319-HAS/IMPACT_open_source/LICENSE-DATA) applies to dataset assets in `dataset/`, repository-authored documentation in `docs/`, the top-level `README.md`, and Markdown documentation under `tasks/`, unless otherwise noted

Third-party materials:
- `third_party/` retains the bundled upstream licenses of the corresponding methods

Precedence:
- if a file or subdirectory contains a more specific license notice, that notice takes precedence over the repository-level defaults
