# Tasks

Benchmark code is organized first by task and then by method.

Current release:
- `TAS/`: `LTContext`, `ASQuery`, `DiffAct`, `FACT`
- `ASR/`: `MS-TCN++`, `VideoMAE v2+Head`
- `PSR/`: `MS-TCN++ -> PSR`, `VideoMAE v2+Head -> PSR`, `STORM-PSR`
- `PPR/`: `ASQuery`, `DiffAct`, `LTContext`, `FACT`
- `ATR/`: `LTContext`, `FACT`
- `CV-TA/`: `Cosine kNN`
- `CV-SM/`: `CV-SMR`, `CV-SMC`

Task directories use the paper abbreviations in uppercase. Method directories use lowercase or snake_case names to keep script paths stable.
