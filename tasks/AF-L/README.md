# Long-Horizon Action Forecasting

This directory reserves the `AF-L` benchmark described in the IMPACT paper.

## Status

- The current public repository provides the `AF-L` task placeholder only.
- Runnable `AF-L` baselines are not released in the current code drop.

## Planned Baselines

- `ScalAnt`
- `AntGPT`
- `PALM`
- `Qwen3VL-8B`

## Notes

- Per the paper, `AF-L` operates on step-level segments derived from `TAS-S`.
- This directory is kept in the public repository so the task layout matches the paper taxonomy.
